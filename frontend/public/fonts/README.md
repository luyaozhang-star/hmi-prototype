# TT Commons Pro Fonts Directory

## Required Font Files

Place your TT Commons Pro TTF font files in this directory with the following exact names:

### TT Commons Pro (Main HMI Font)
- ✅ **TT-Commons-Pro-Regular.ttf** (400 weight) - Body text
- ✅ **TT-Commons-Pro-Medium.ttf** (500 weight) - Medium weight text
- ✅ **TT-Commons-Pro-DemiBold.ttf** (600 weight) - Headings & emphasis

### TT Commons Pro Mono (Cluster Display Font)
- ✅ **TT-Commons-Pro-Mono-Regular.ttf** (400 weight)
- ✅ **TT-Commons-Pro-Mono-DemiBold.ttf** (600 weight)

## All Files Are Required

All 5 font files listed above should be placed in this directory for the HMI to display properly.

## After Adding Fonts

1. Place the TTF files in this directory
2. Restart the dev server if it's running
3. Clear browser cache and refresh
4. The fonts should now load properly!

## Verification

To verify fonts are loaded:
1. Open browser DevTools → Network tab
2. Filter by "Font"
3. You should see the TTF files loading
4. Check Console for any font loading errors
